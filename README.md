## 搬瓦工到底是个什么商家

搬瓦工的英文品牌名是 BandwagonHost，母公司 IT7 Networks，2004 年就在跑这个生意，算是老牌 VPS 商家里少有的还活着、还越活越滋润的那批。它最早靠"便宜大碗"起家，5 美元一年那种 OVZ 套餐当年养活了一大批折腾党。后来 OVZ 下架，普通 KVM 也慢慢往中国优化线路上靠，现在主打的就是 CN2 GT、CN2 GIA-E、香港/大阪/新加坡/东京 CN2 GIA 这几条针对国内用户优化过的线路。

它的管理面板叫 KiwiVM，自己开发的，能开机、关机、重装系统、迁机房、做快照、查流量、改 rDNS，最近还加上了 AMD EPYC + NVMe RAID-10 的新机型，纽约、洛杉矶 DC9、香港 HK3/HK8 都陆续换装。系统模板覆盖 AlmaLinux、RockyLinux、CentOS、Debian、Ubuntu、Fedora，新出的 Debian 13、Ubuntu 26.04 也都跟进上了。

简单说一句：它不是最便宜的 VPS，但在中国线路这块，它一直是"贵得有道理"的那个。

## 关于"中文官网"这件事，先说句实话

搬瓦工的真正官方网站是英文界面，没有官方的"中文官网"——你搜到的那一堆带"中文""笔记""指南"字样的站点，本质上都是第三方整理站，做的是教程、优惠码、套餐翻译、补货通知这些活儿。它们有用，但不是官网，下单付款还是要跳回英文主站完成。

所以"搬瓦工中文官网"这个搜索意图背后，真正想问的是三件事：

- **去哪下单才不会被假站骗**——认准 bandwagonhost.com 这个主域名，付款页面有 SSL，支持支付宝、微信、PayPal、信用卡、银联。
- **看不懂英文界面怎么办**——下单流程其实就四步：选套餐 → 选时长和机房 → 填优惠码 → 注册账号付款。后面会一步步讲。
- **优惠码、套餐信息在哪看**——第三方整理站更新快，但价格、配置这些硬数据以官网为准，下面给的表格就是按官网当下在售方案整理的。

## 关于优惠码，说几句实在话

搬瓦工的优惠码生态比较特别：常年的常规码（BWH 开头那批）折扣基本在 4.4%–6% 之间，6% 已经是天花板。2026 年初出现的 `NODESEEK2026` 折扣 6.77% 算是少见的高位，但这种"高折扣码"通常活不长，能不能赶上全凭运气。

目前在各大整理站还能看到的几个码：

- `BWH1ZBPVK` —— 6%，常规码里折扣最高的
- `BWH1XZOBK` —— 5.5%
- `ireallyreadtheterms8` —— 5.5%
- `BWH1NJJHL` —— 4.5%
- `ireadtheterms8` —— 4.4%

大促节点（双十一、黑五、元旦、年中年末）会有临时大额码或者直接降价，力度比常规码高一截。如果你不急，等大促再下单能省一笔；如果急用，随便用上面任意一个 5.5% 以上的码就行，反正结账时填进去自动算。

> 一个小提醒：优惠码在新购时可用，但续费通常不能叠加码（搬瓦工的续费本身就是原价续费，不会涨价）。所以下单时把年限选长一点更划算——年付比月付单价低，再加上优惠码，差价就出来了。

## 注册和购买流程（支付宝版，四步走）

虽然官网是英文，但下单流程其实非常简单，把它当淘宝走就行：

**第一步：选套餐。** 上面的套餐表里点 👉 购买链接，会直接跳到对应套餐的下单页，不用自己再去翻菜单。

**第二步：选时长和机房。** 页面上会有 Billing Cycle（计费周期）下拉框，月付 / 半年付 / 年付 / 两年付自由选；下面是机房位置（Location），CN2 GIA-E 套餐默认是 DC6，可以下单后在 KiwiVM 面板里免费一键切换到 DC9、JPOS_1、EUNL_9 等其他可用机房。普通 KVM 套餐可选机房更多，9 个美国机房加欧洲、加拿大都能切。

**第三步：填优惠码。** 在 Order Summary 那一栏有 Promotional Code 输入框，把上面的码粘进去，点 Validate Code，金额会自动减。然后点 Continue 进入下一步。

**第四步：注册账号 + 付款。** 第一次购买会让你填邮箱、设密码、填个人信息（地址随便填个英文地址就行）。然后到 Payment Method 选 Alipay（支付宝）或者 WeChat Pay、PayPal、信用卡、银联。选支付宝会跳出一个二维码，手机扫一下就完事，秒到账。订单完成后，KiwiVM 面板的登录链接会发到你邮箱，进去就能看到自己的 VPS，开机状态、IP、root 密码都在那儿。

整个流程下来，英文界面其实没几个生词：Billing Cycle、Location、Promotional Code、Continue、Checkout，认得这五个就能买。

## 几个新手容易踩的坑

最后说几个搜"搬瓦工中文官网"的人最容易栽的地方：

- **认准主域名**。bandwagonhost.com 是唯一的官方主站，其他带"中文""指南""笔记"字样的都是第三方整理站，可以做信息参考，但下单别在那些站里完成，要确认最终付款页是 bandwagonhost.com 的域名。
- **优惠码不是越大越好**。6% 是常规码的天花板，看到宣传"9 折""8 折"的多半是过期码或者骗点击的。下单前在结账页 Validate 一下，能减才是真的。
- **CN2 GT 和 CN2 GIA 不是一回事**。GT 是半程优化（去程走 CN2，回程走普通线路），GIA 才是全程优化。搬瓦工现在主推的 CN2 GIA-E 就是 GIA 增强版，比 GT 贵是有道理的。如果你看到便宜得不正常的"CN2 套餐"，先确认是 GT 还是 GIA。
- **机房可以切换，但流量不一定跟着走**。CN2 GIA-E 套餐在 DC6/DC9/JPOS_1/EUNL_9 之间切换流量不变，但迁到普通机房（DC2、DC4 等）流量会按比例缩水，迁之前在 KiwiVM 面板看清楚提示。
- **续费不能叠加优惠码**。所以下单时把年限选长一点更划算，年付单价本来就比月付低，再加优惠码，差价就出来了。
